## Sign up with a personal Microsoft account

1. Go to [Azure DevOps](https://azure.microsoft.com/services/devops/) and select **Get started with Azure**.
2. Sign in with your Microsoft account. If you don't have one, [create a Microsoft account](https://azure.microsoft.com/services/devops/).
3. Select either **Try Azure for free** or **Pay as you go** and complete the sign-up process.
4. In the Azure portal, select the app launcher (grid icon), and then select **DevOps**.
5. Select **Create new organization**.

After sign-up, access your organization at any time: `https://dev.azure.com/{Your_Organization}`.
