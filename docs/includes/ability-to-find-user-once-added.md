> [!NOTE]
> You might not be able to find a user from a permissions page or identity field if the user wasn't added to the project either by adding it to a security group or to a project team. Also, when a user is added to Microsoft Entra ID or Active Directory, there can be a delay between the time when they're added to the project and when they're searchable from an identity field. The delay can be between 5 minutes to 7 days.
