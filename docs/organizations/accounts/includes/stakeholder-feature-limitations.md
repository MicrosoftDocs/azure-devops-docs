Users with Stakeholder access can sign in and work with boards (view and update work items, comments, and dashboards), but they can't access private Git repositories, including viewing code, cloning repos, pushing changes, or reviewing pull requests.

Navigation doesn't include **Repos** or **Test Plans** hub:

:::image type="content" source="../media/faq/navigation-no-repos-test-plans.png" alt-text="Screenshot showing navigation with Overview, Dashboards, Wiki, Boards, and Pipelines but no Repos or Test Plans hub.":::

Accessing **Repos** or **Test Plans** directly returns a 404 error:

:::image type="content" source="../media/faq/repos-test-plans-404-return.png" alt-text="Screenshot showing a 404 error page when directly accessing Repos or Test Plans with Stakeholder access.":::
