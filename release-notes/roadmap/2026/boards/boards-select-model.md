# Select a model for Copilot Cloud Agent integration

Enhance the [Copilot integration with Azure Boards](/azure/devops/release-notes/2026/boards/sprint-268-update#general-availability-of-github-copilot-integration-for-azure-boards) by allowing users to select from available models. This enables users to choose an LLM that best fits their coding agent and repository needs.
