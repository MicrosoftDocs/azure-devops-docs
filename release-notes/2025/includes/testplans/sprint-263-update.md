### Tester column renamed in test points to Current Tester

To improve clarity regarding test point assignments, the column previously labeled Tester has been renamed to Current Tester. This change is purely cosmetic and does not affect any underlying system logic or functionality.

### Fixed query results pane resizing issue

Resolved a defect preventing the query and results area from being resized during requirements-based suite creation.

### Fixed missing screenshots for individual steps

Resolved a defect causing inconsistent screenshot capture in individual steps when using the Test and Feedback Extension.
