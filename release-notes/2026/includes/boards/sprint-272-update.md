### Improved markdown editor usability in work item multi-line fields

Markdown multi-line fields in work items now separate preview and edit modes more clearly. Fields stay in read mode by default, making it easier to review content, follow links, and avoid accidental edits.

When you want to update the field, you can switch into edit mode intentionally instead of entering edit mode through incidental actions such as scrolling or clicking the container.

### Filter Boards and backlogs by additional fields

You can now filter Azure Boards and Backlogs by additional fields, making it easier to narrow large board and backlog views without creating a new query.

This update also improves filtering scenarios such as parent values and longer numeric field values.
