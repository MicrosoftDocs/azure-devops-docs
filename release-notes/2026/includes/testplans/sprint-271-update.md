### Query and relate work items across projects in test results

When analyzing test results, you may need to search for and relate bugs or other work items that live in a different project than the test result itself.
Now we support this scenario by allowing you to query and relate work items across projects. To use this capability, enable the **Query across projects** option when searching for work items from a test case result.
